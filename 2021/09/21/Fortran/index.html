输入输出

  1. write 和 print 的区别
    write 的基本格式：
    write(*,*)var
    print的基本格式：
    print *,var
    print 只有一个星号，用来指定输出格式，缺少的另一个星号为输出位置，print只能输出到屏幕，因此无输出位置选项。其他用法基本一致。
  2. 格式化输出
    write(*,*)的标准写法
    write(UNIT=*,FMT=*)
    其中UNIT用来指定输出位置，FMT用来指定输出格式。
    write的另一种表示方式： 程序中，使用行代码表示格式，也就是用第二行的格式输出变量a,I4表示只会用4个字符宽度输出整数。Fortran 中的Format包括Iw、Aw、Fw.d、nX等。

– 格式化说明
write(*,"(I5)") 100
以5个字符输出整数，至少输出5个数字，不足的前面补空格。
write(*,"(I3)") 1000
以3个字符输出，但是长度不够，结果显示***作为警告。
write(*,"(I5.3)") 10

  • 判断语句 注意：if (4>3>2)是不对的，必须写成 if (4>3 .and. 3>2 ). 因为逻辑运算从左到右执行，4>3结果为logical类型=1，并不大于2。
